= MoneyBook — 定数定義書
:toc: left
:toclevels: 2

== 概要

本書はソースコード中で定義・使用される定数をすべて列挙する。

== app-constants.js

=== INCOME_CATEGORIES

収入カテゴリの文字列配列。カテゴリ選択肢・バリデーションに使用する。

[source,javascript]
----
['給与', 'ボーナス', '副収入', '投資', 'その他収入']
----

=== EXPENSE_CATEGORIES

支出カテゴリの文字列配列。カテゴリ選択肢・バリデーションに使用する。

[source,javascript]
----
['食費', '交通費', '光熱費', '通信費', '娯楽', '医療', '衣類', '日用品', 'その他支出']
----

=== STORAGE_KEYS

LocalStorage キー名のマップ。

[cols="2,2,3"]
|===
| プロパティ | 値 | 格納内容

| `RECORDS` | `'moneybook_records'` | 収支レコードの JSON 配列
| `BUDGETS` | `'moneybook_budgets'` | 予算設定の JSON オブジェクト
|===

=== CSV_HEADERS

CSV エクスポート時のヘッダー行文字列配列。

[source,javascript]
----
['日付', '種別', 'カテゴリ', '金額', 'メモ']
----

=== RECORD_TYPES

収支の種別識別子マップ。

[cols="2,2,3"]
|===
| プロパティ | 値 | 説明

| `INCOME`  | `'income'`  | 収入レコード
| `EXPENSE` | `'expense'` | 支出レコード
|===

== code-definitions.js

=== MODAL_STATE

モーダルの表示状態コードマップ。

[cols="2,2,3"]
|===
| プロパティ | 値 | 説明

| `CLOSED` | `'closed'` | モーダル非表示
| `ADD`    | `'add'`    | 追加モーダル表示中
| `EDIT`   | `'edit'`   | 編集モーダル表示中
|===

=== ERROR_CODES

バリデーションエラーコードマップ。`validateRecord()` / `validateBudget()` の戻り値に使用する。

[cols="2,2,3"]
|===
| プロパティ | 値 | 発生条件

| `INVALID_AMOUNT`   | `'INVALID_AMOUNT'`   | 金額が正の整数でない、または未入力
| `INVALID_DATE`     | `'INVALID_DATE'`     | 日付が `YYYY-MM-DD` 形式でない、または未入力
| `MISSING_CATEGORY` | `'MISSING_CATEGORY'` | カテゴリが候補リストに含まれない、または未選択
| `INVALID_TYPE`     | `'INVALID_TYPE'`     | 種別が `income` / `expense` 以外
|===
