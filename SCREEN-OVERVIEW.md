# MoneyBook — 画面構成説明

## 構成概要

アプリは単一ページ（SPA）として動作する。`index.html` 上にすべての UI を持ち、モーダルで入力フォームを表示する。

---

## ダッシュボード（index.html）

### ヘッダー

| 要素 | 内容 |
|------|------|
| ロゴ | "Money**Book**"（Book 部分はミントグリーン） |
| 月ナビゲーション | ‹ ボタン / YYYY年MM月 / › ボタン |

前月・翌月ボタンをクリックすると、サマリー・一覧・チャートがすべて当月分に更新される。

---

### サマリーカード（3 枚）

| カード | 表示値 | 色 |
|--------|-------|----|
| 収入 | 当月の収入合計 | ミントグリーン |
| 支出 | 当月の支出合計 | コーラルレッド |
| 残高 | 収入 − 支出 | シアン |

---

### チャートセクション

- 支出カテゴリ別のドーナツチャート（Chart.js）
- 凡例はチャート下部に配置
- 当月の支出が 0 件の場合は「支出データがありません」というテキストを表示し、チャートは非表示

---

### 収支一覧

- 当月の全レコードを日付降順で表示
- 各行の構成：カラードット（種別色）/ カテゴリ名 / 日付・メモ / 金額（符号付き）
- レコードをクリックすると編集モーダルが開く
- 当月の記録が 0 件の場合は「記録がありません」を表示
- 「CSV 出力」ボタンで当月データを `moneybook_YYYY-MM.csv` としてダウンロード

---

### FAB ボタン（＋）

- 画面右下に固定表示のミントグリーンの丸ボタン
- クリックで追加モーダルを開く

---

## 追加 / 編集モーダル

| フィールド | 種類 | 制約 |
|-----------|------|------|
| 日付 | date input | 必須・YYYY-MM-DD 形式 |
| 種別 | radio（支出 / 収入） | 必須 |
| カテゴリ | select | 種別に連動して選択肢が切り替わる |
| 金額（円） | number input | 必須・正の整数のみ |
| メモ | text input | 任意・最大 100 文字 |

- **追加モード**：「キャンセル」「追加」ボタンを表示
- **編集モード**：「削除」「キャンセル」「更新」ボタンを表示
- バリデーションエラー時はフォーム下部にエラーメッセージを赤字で表示
